# 운영체제란?

## 정의
운영체제(OS)는 하드웨어와 사용자 사이의 **중재자 역할**을 하는 시스템 소프트웨어이다.

## 핵심 역할
- **자원 관리**: CPU, 메모리, I/O 장치 등을 효율적으로 관리
- **추상화**: 복잡한 하드웨어를 단순한 인터페이스로 제공
- **보호**: 프로세스 간 충돌 방지, 사용자 권한 관리

## 커널(Kernel)
OS의 핵심 부분으로 하드웨어와 직접 통신
- 프로세스 스케줄링
- 메모리 관리
- 파일 시스템 관리

## 시스템 콜
사용자 프로그램 → OS 기능 요청 인터페이스
```c
int fd = open("file.txt", O_RDONLY); // 파일 열기 시스템 콜
```

> 💡 OS는 하드웨어 자원을 관리하고 사용자와 하드웨어 사이의 다리 역할을 한다.
