# 컴퓨터 하드웨어 구조

## 주요 구성 요소

### CPU
- 컴퓨터의 두뇌, 명령어 해석 및 실행
- ALU(산술논리장치) + CU(제어장치) + 레지스터

### 메모리 (RAM)
- 실행 중인 프로그램/데이터 임시 저장
- 휘발성: 전원 꺼지면 소멸

### 저장장치
| 종류 | 특징 |
|------|------|
| HDD | 자기 디스크, 느리고 저렴 |
| SSD | 플래시 메모리, 빠르고 내구성↑ |

### 버스 (Bus)
컴포넌트 간 데이터 통신 경로
- 데이터 버스 / 주소 버스 / 제어 버스

## 메모리 계층 구조
레지스터 (가장 빠름)
↓
캐시 (L1 → L2 → L3)
↓
RAM
↓
SSD/HDD (가장 느림, 용량 큼)

> 💡 속도가 빠를수록 용량은 작고 비용은 비싸다 (트레이드오프)
